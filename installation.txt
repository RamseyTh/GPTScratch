Python 3.10 or 3.11 recommended.

Install dependencies:
pip install -r requirements.txt

Required model files:
model/model_weights.pt
model/hftokenizer/

Raw filings:
Place Form 10-K files under data/raw/.

Verified questions:
Place or install verified questions under data/questions/.

Build chunks:
python scripts/build_corpus.py \
  --data-dir data/raw \
  --out outputs/chunks/chunks.jsonl

Run main experiment:
python main.py --quick-run --run-id final_verified_run --report

Smoke test:
python main.py --quick-run --limit 3 --run-id smoke_test --report

No OpenAI API key is required.
